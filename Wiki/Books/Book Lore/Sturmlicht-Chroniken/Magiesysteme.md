# Magiesysteme der Sturmlicht-Chroniken

> „Die wahren Erschütterungen sind sowohl von Ehre als auch von Kultivierung — Kultivierung für das Leben, Ehre, um die Erschütterung zum Naturgesetz zu machen."
> — Raboniel

---

## 1. Surgebinding (Bogensprengung)

**Surgebinding** ist die vorherrschende Investitur-Manifestation auf Roschar. Surgebinder manipulieren zehn fundamentale Kräfte — die **Erschütterungen** (*Surges*) — indem sie Objekte oder Wesen mit **Sturmlicht** infundieren.

### Die Zehn Erschütterungen

| # | Erschütterung | Wirkung | Orden |
|---|--------------|---------|-------|
| 1 | **Adhäsion** | Dinge aneinander haften lassen / atmosphärische Bindung | Windläufer |
| 2 | **Gravitation** | Schwerkraft manipulieren (fliegen, Fallen lenken) | Windläufer, Himmelsbrecher |
| 3 | **Teilung** | Materie auflösen / zerstören | Himmelsbrecher, Staubbringer |
| 4 | **Abrasion** | Reibung reduzieren (rutschen, gleiten) | Staubbringer, Kantentänzer |
| 5 | **Fortschritt** | Heilung, Wachstum beschleunigen | Kantentänzer, Wahrheitswächter |
| 6 | **Illumination** | Lichtbrechung, Illusionen erschaffen | Wahrheitswächter, Lichtweber |
| 7 | **Transformation** | Stoffe in andere Stoffe verwandeln (Gspräch mit der Seele) | Lichtweber, Seherufer |
| 8 | **Transport** | Zwischen den Reichen reisen (Physisch ↔ Kognitiv) | Seherufer, Willensformer |
| 9 | **Kohäsion** | Materie molekular „weich machen" / formen | Willensformer, Steinwächter |
| 10 | **Spannung** | Materie härter / elastischer machen | Steinwächter, Bindungsbrecher |

### Wie wird man Surgebinder?

| Methode | Beschreibung |
|---------|-------------|
| **Ehrenklinge** | Heilige Waffe der Herolde — gibt sofortige vollen Zugang (aber keine Nahel-Bindung) |
| **Nahel-Bindung** | Verbindung mit einem Erhabenen Spren → Stufenweise Kräfte durch Eide |
| **Verschmolzene (Fused)** | Singende, die von Odium mit Kräften erfüllt wurden → ewige Seelen in neuen Körpern |
| **Direkter Segen** | Seltene direkte Gabe einer Splittergottheit |
| **Morgenrötesänger (Dawnsingers)** | Uralte, fast vergessene magische Rasse |

## 2. Sturmlicht

**Sturmlicht** (*Stormlight*) ist die Energieform auf Roschar — gewaltige, bläulich-weiße Energie, die während der Hochstürme freigesetzt wird. Es kann in **Edelsteinen** gespeichert werden:

| Edelstein | Reinheit | Kapazität |
|-----------|----------|-----------|
| Diamant | Makellos, gut, mittel, schwach | Höchste |
| Saphir | Makellos… | Verschiedene |
| Amethyst | … | … |
| Rubin | … | Speichert Wärme (Wärmerubine) |
| Smaragd | … | nährt Pflanzen |
| Topas | … | — |
| Granat | … | — |

Je reiner der Edelstein, desto besser hält er Sturmlicht.

> **Fabriale**: Roscharanische Technologie, die mit eingefangenem Sturmlicht betrieben wird — von einfachen Wärmelaternen bis zu komplexen Maschinen.

## 3. Nahel-Bindung

Die **Nahel-Bindung** ist die Seelenverbindung zwischen einem Menschen und einem **Erhabenen Spren**. Diese Bindung verleiht dem Menschen surgebinding-Fähigkeiten, die mit jedem gesprochenen **Ideal** stärker werden.

### Die Ideale der Strahlenden Ritter

| Ideal | Wirkung |
|-------|---------|
| **1. Ideal** | „Leben vor dem Tod, Stärke vor der Schwäche, Reise vor dem Ziel." → Basis-Zugang |
| **2. Ideal** | Individuell für jeden Orden → Waffenform (Shardklinge) |
| **3. Ideal** | → Shardrüstung (Shardplatte) |
| **4. Ideal** | Erweiterte Kräfte |
| **5. Ideal** | Höchste Stufe → Vollkommene Vereinigung mit Spren |

### Orden & Spren

| Orden | Spren | Attribut(e) | Erschütterungen |
|-------|-------|-------------|-----------------|
| Windläufer | Ehrenspren | Beschützen / Führen | Adhäsion, Gravitation |
| Himmelsbrecher | Hochspren | Gerecht / Selbstbewusst | Gravitation, Teilung |
| Staubbringer | Aschspren | Mutig / Gehorsam | Teilung, Abrasion |
| Kantentänzer | Kulturspren | Liebend / Heilend | Abrasion, Fortschritt |
| Wahrheitswächter | Nebelspren | Gelehrt / Gebend | Fortschritt, Illumination |
| Lichtweber | Kryptik (Lügenspren) | Kreativ / Ehrlich | Illumination, Transformation |
| Seherufer | Tintenspren | Weise / Vorsichtig | Transformation, Transport |
| Willensformer | Reacher (Lichtspren) | Entschlossen / Bauend | Transport, Kohäsion |
| Steinwächter | — | Fromm / — | Kohäsion, Spannung |
| Bindungssprecher | Stormvater / Geschwister / Nachtwächter | — | Nur Adhäsion (besondere Rolle) |

## 4. Die Rhythmen der Singenden

Die **Singenden** (*Listeners / Parshendi*) nutzen **Rhythmen** statt Sturmlicht. Sie existieren im Einklang mit den natürlichen Tönen Roschars. Jeder Rhythmus hat eine emotionale und kulturelle Bedeutung:

| Rhythmus | Emotion/Zweck |
|----------|--------------|
| Rhythmus der Ruhe | Meditation, Frieden |
| Rhythmus der Freude | Feierlichkeiten |
| Rhythmus der Erregung | Kampf, Spannung |
| Rhythmus der Ablehnung | Widerstand, Wut |
| Rhythmus der Furcht | Angst |
| **Rhythmus der Vernichtung** | → Der zerstörerische Rhythmus Odiums |

Durch bestimmte Rhythmen können die Singenden **Sturmlichter** anziehen oder **Voidlight** (Odiums Energie) kanalisieren.

## 5. Shardklingen & Shardrüstungen

**Shardklingen** (*Shardblades*) sind magische Waffen, die durch den Geist beschworen werden können:

| Eigenschaft | Shardklinge | Shardrüstung |
|-------------|-------------|-------------|
| Erscheinung | 1,80 m langes Großschwert | Vollkörperpanzer |
| Gewicht | Praktisch nichts | Leicht für Träger |
| Schlagwirkung | Schneidet durch jede Materie | Unzerstörbar |
| Seelen-Schock | Kappt die Seele (tödlich für Lebewesen) | — |
| Spren | Beschworen aus Nahel-Bindung | Wird aus Spren-Materie gebildet |
| Regeneration | Über Stunden bis Tage | Schnell (mit Sturmlicht) |
