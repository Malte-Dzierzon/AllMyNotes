# Wichtige Ereignisse (Arc 1)

> Die Zusammenfassung der zentralen Ereignisse der ersten fünf Bände der Sturmlicht-Chroniken.

| Ereignis | Buch | Bedeutung |
|----------|------|-----------|
| **Gavilars Tod** | TWoK | Startet die Handlung — Szeth's erster Mord |
| **Kaladin wird Brückenläufer** | TWoK | Hero Journey beginnt |
| **Brücke Vier wird gerettet** | TWoK | Kaladin spricht das 1. + 2. Ideal |
| **Schallan findet Urithirus Karte** | WoR | Die Turmstadt wird wiederentdeckt |
| **Eshonai wechselt zur Sturmform** | WoR | Die Listener werden zu Hervorbringern |
| **Die Schlacht an den Zerschmetterten Ebenen** | WoR | Die Alethi siegen — Urithiru wird erreicht |
| **Szeths Tod und Wiederkehr** | WoR | Getötet von Kaladin → zu Nales Himmelsbrecher |
| **Kholinar fällt** | OB | Die Hauptstadt geht verloren |
| **Schlacht von Thaylen Field** | OB | Dalinar besiegt Odium (temporär) |
| **Die Ungeformte Sja-anat rebelliert** | OB | Wendepunkt für die Spren |
| **Urithiru besetzt** | RoW | Die Verschmolzenen unter Raboniel besetzen die Stadt |
| **Anti-Sturmlicht entdeckt** | RoW | Navani versteht die Töne der Investitur |
| **Kaladins 4. Ideal** | RoW | "Ich werde diejenigen akzeptieren, die ich nicht retten kann" |
| **Tefts Tod** | RoW | Von Moash getötet — emotionaler Tiefpunkt |
| **Venli spricht das 1. Ideal** | RoW | Erste Singende Strahlende |
| **Dalinar wird zum Gefäß** | WaT | Kurzzeitig Gefäß von Ehre → lehnt es ab |
| **Taravangian wird Todium** | WaT | Odiums Gefäß wechselt |
| **Kaladin steigt zum Herold auf** | WaT | Er wird der 10. Herold |
| **Rachsucht entsteht** | WaT | Ehre + Odium verschmelzen |
| **Zeitdilatation um Roschar** | WaT | ~70-80 Jahre vergehen fürs Kosmeer |
