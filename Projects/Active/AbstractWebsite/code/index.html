PHILOSOPHY

DIGITAL RUINS

The website lives in the tension between organic and digital. Never fully organic, never fully digital — always in the space between.

Texture is the primary visual language. Where other designs use color variation, this system uses grain, noise, scanlines, halftones, chromatic aberration, and paper texture. The palette is intentionally narrow — visual richness comes from texture variation, not chromatic variety.

CORE PRINCIPLES

SIX AXIOMS

  1. 01 Texture Before Color — Visual richness from grain, noise, scanlines, halftones.
  2. 02 Imperfection as Identity — Glitches, pixelation, distortion are the visual signature.
  3. 03 Asymmetry as Order — 60/40 split, diagonal division, off-balance composition.
  4. 04 Typography as Material — Text is texture, shape, pattern, atmosphere.
TEXTURE LAYERS

THE SOUL OF THE DESIGN

LAYER 1 · MANDATORY

FILM GRAIN

Fractal noise, 0.65 baseFrequency, 4 octaves. Opacity 3-18% by context. Breathing animation 8s cycle.

LAYER 3 · OPTIONAL

CRT SCANLINES

Repeating linear gradient, 3px repeat. Subtle (4%), heavy (10%), or cyan-tinted variants. CRT monitor phosphor feel.

LAYER 4 · SIGNATURE

CHROMATIC ABERRATION

RGB channel separation (±2px). Cyan/magenta channels. Core visual texture, not hover gimmick. 4s oscillation cycle.
