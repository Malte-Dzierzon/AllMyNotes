# MTG Multiverse Studio

**A local-first Magic: The Gathering desktop app** — collection manager, deck builder, Artbook & lore-narrative. No cloud, no tracking.

## Tech Stack
- **Frontend:** TypeScript, React, Vite
- **Backend:** Rust (Tauri)
- **Architektur:** Desktop-native (Tauri), lokal-first
