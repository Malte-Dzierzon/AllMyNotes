## About

This is my personal Obsidian vault. It serves as:

- **Second Brain** — structured knowledge for me and AI agents
- **Project Dashboard** — overview of active and archived projects
- **Research Hub** — automated research collection via cron agents
- **GitHub Mirror** — synced across devices (desktop + tablet)

---

## Structure

```
AllMyNotes/
├── Wiki/          Second Brain (Books, Images, Notes)
├── Projects/      Active & archived project folders
├── Research/      Automated research inbox + topics
├── Excalidraw/    Excalidraw drawings (plugin-managed)
├── _Meta/         Templates, scripts, vault infrastructure
└── README.md      This file
```
